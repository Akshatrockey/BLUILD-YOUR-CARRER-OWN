# CareerPathway - Career Guidance Platform for College Students

A comprehensive web-based platform helping college students discover career paths, create personalized roadmaps, and connect with experienced mentors.

## 🌟 Features

- **Career Exploration**: Browse 100+ career paths across different fields
- **Personalized Roadmaps**: Get step-by-step preparation plans with milestones
- **Expert Mentorship**: Connect with industry professionals for guidance
- **Rich Resources**: Access articles, courses, and tools for every career
- **Progress Tracking**: Monitor your journey with interactive checklists
- **Community Network**: Connect with students pursuing similar careers

## 🎯 Career Fields Covered

- **Technology**: Software Development, Data Science, AI/ML Engineering
- **Business**: Management, Finance, Entrepreneurship
- **Healthcare**: Medicine, Nursing, Public Health
- **Engineering**: Civil, Mechanical, Electrical
- **Creative Arts**: Design, Animation, Content Creation
- **Science**: Research, Academia, Environmental Science

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code, Sublime Text, etc.)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Akshatrockey/CareerPathway.git
cd CareerPathway
```

2. Open `index.html` in your browser:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server

# Or simply double-click index.html
```

3. Visit `http://localhost:8000` in your browser
